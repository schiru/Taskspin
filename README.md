# Taskspin
Taskspin, a quick and easy task manager.

## Features: 
* task nesting
* heavy use of keyboard shortcuts
* offline functionality

Note: This project is no longer being actively developed.

## Technologies used:
JavaScript, jQuery, PHP, Appcache, localStorage
